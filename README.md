# Louisiana Child — Official Site

The official website for Louisiana Child, a Toronto rock band.

**Live:** https://louisianachild.ca

## Stack

Single static HTML file. No build step. No dependencies. Deployed on Vercel.

## Files

- `index.html` — the entire site (HTML + inline CSS, no JS)
- `hero-bg.jpg` — hero background photo
- `logo.png` — wordmark logo
- `favicon-32.png` — browser tab icon
- `apple-touch-icon.png` — iOS home-screen icon
- `vercel.json` — security headers config

## Local preview

Just open `index.html` in any browser.

## Deployment

Push to `main`. Vercel auto-deploys.
